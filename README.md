# 🚀 Infrastructure & CI/CD: Products Microservice

This repository handles the automation and cloud setup for the **Products** service. It is part of the **Cloud-Native eCommerce App** project.

### 🛠️ What's inside:
- **CI/CD Pipelines**: Azure DevOps YAML definitions for automated build, unit testing, and Docker image creation.
- **Orchestration**: Kubernetes manifests and configuration for deployment on **Azure Kubernetes Service (AKS)**.
- **Tech Stack**: MySQL + Entity Framework Core.
